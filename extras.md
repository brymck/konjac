## Dictionaries

Some free dictionaries for full-width/half-width conversion are available for
download via the following commands:

{% highlight bash %}
git clone git://github.com/brymck/konjac_yml.git
cp konjac_yml/*.yml ~/.konjac
{% endhighlight %}
